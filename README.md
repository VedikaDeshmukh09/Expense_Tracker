# 💰 Expense Tracker Application

A desktop-based Expense Tracker application developed using **Java, Spring Boot, JavaFX, and MySQL**. The application helps users manage their daily expenses by recording transactions, categorizing spending, and viewing expense history through an easy-to-use interface.

## 🚀 Features

- Add daily income and expenses
- Categorize transactions
- View all expense records
- Store data securely in MySQL database
- User-friendly JavaFX interface
- CRUD operations (Create, Read, Update, Delete)
- Simple and responsive design

## 🛠️ Tech Stack

- Java
- Spring Boot
- JavaFX
- MySQL
- JDBC
- Maven
- IntelliJ IDEA

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ExpenseTracker.git
```

2. Open the project in IntelliJ IDEA.

3. Create a MySQL database.

4. Import the SQL file (if provided).

5. Update database credentials in the configuration file.

6. Run the application.
